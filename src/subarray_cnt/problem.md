# Problem

Given an array `arr` of integers, the MEX of any of its subarrays can be between 0 and n, inclusive. Return, for each value `v` from 0 to n, the number of subarrays of `arr` whose MEX is equal to `v`.

# Example

`arr = [0, 3, 0, 1, 0, 1, 5, 4, 2, 0]`.  
Then we return `[12, 10, 22, 7, 0 , 0, 4, 0, 0, 0, 0]`
